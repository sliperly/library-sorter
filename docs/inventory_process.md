## Сканирование файлов
```bash
find $LIBRARY_ROOT -maxdepth 2 -type d | sort
find $LIBRARY_ROOT -type f \
  -not -path "*/lost+found/*" \
  -printf "%s\t%TY-%Tm-%Td\t%p\n" \
  | sort -k3 > library_inventory.tsv
```
